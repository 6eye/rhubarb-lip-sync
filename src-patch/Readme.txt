These files are for Papagayo format output

1. Copy all the patched files to somewhere.
2. Run 'git checkout 8d2d1003769889b7488016c9e67f48b1c58ec6f9', this command will checkout the version it based.
3. Replace source files with the patched files.
4. Run the build script to build all.

Mouth shapes:
http://www.heliumfrog.com/Helium_Frog/helium_frog_documentation/helium_frog_exposure_sheet/heliumfrogexposuresheet.html

I use Papagayo format, for blender has a plug-on to import Papagayo format files.
